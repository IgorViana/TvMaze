## Objetivo
Teste para empresa Afya, criação de apicativo utilizando tecnologias atuais.

## O que foi entregue

- Lista de Séries
- Busca por séries
- Detalhes de séries
- Lista de episodios por temporada
- Detalhe de episodios

## Tecnologias usadas

- Kotlin
- Viewbinding
- Navigation
- Coroutines
- MVVM
- Koin
- Retrofit
- Glide
- Paging
